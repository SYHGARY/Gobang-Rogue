# Gobang-Rogue
This game is written in C++. The core gameplay is a Gobang game that contains Roguelike elements and growth elements. The core technology includes AI algorithms and national security communication algorithms.

版本：Alpha测试版1.0

最后更新日期：2025年12月11日

作者：GARYSONG

一、	项目概述
1.	项目名称：Gobang Rogue
2.	项目愿景：打破传统五子棋的定式博弈，通过随机性、成长性和策略深度，为玩家带来“每局都是新体验”的沉浸式对战乐趣。
3.	核心特色：暂定
4.	技术栈：
   
  （1）	核心开发语言：C++（用于游戏逻辑、渲染、计算）

  （2）	测试语言：Python（AI算法测试、数据分析）

  （3）	数据库：MySQL（用于在线玩家数据、排行榜）

  （4）	网络通信：基于C++套接字

  （5）	图形界面：基于C++的EasyX库

1.	基础功能：
   
（1）	游戏核心：

5.	Roguelike元素功能：
   
  （1）	卡牌池：设计一系列改变规则的技能卡牌。

  （2）	获取方式：每N回合从卡牌池中随机挑取三张卡牌，供玩家选择其一。同时，若玩家达成特定条件（待考虑），获得特殊卡牌。

  （3）	使用规则：每回合要使用卡牌最多只能使用一张。卡牌获取有上限。
